## Algorithm:

### Step 2: Enter the elements of the matrix

### Step 4: Print the rank of the matrix

## Program:
~~~

import numpy as np
A=np.array([[1,0,3],[-1,2,-2],[2,3,-1]])
sol=np.linalg.inv(A)
